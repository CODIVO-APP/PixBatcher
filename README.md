## ‚ú® FEATURES

### Core Functionality
- **Batch Image Conversion** - Convert multiple images at once
- **50+ Format Support** - WebP, PNG, JPEG, SVG, PDF, PSD, TIFF, BMP, ICO, and more
- **Modern Format Support** - HEIC/HEIF, AVIF, WebP with advanced compression
- **RAW Image Support** - Process RAW files from professional cameras
- **PDF Conversion** - Convert PDFs to images or images to PDF
- **Quality Control** - Adjustable quality settings for optimal file size
- **Preserve Metadata** - Keep EXIF data during conversion
- **Multi-threaded Processing** - Fast conversion using parallel processing
- **Drag & Drop Interface** - Easy file selection
- **Progress Tracking** - Real-time conversion status

### Supported Input Formats
- **Standard:** JPEG, PNG, BMP, TIFF, GIF, ICO, WebP
- **Advanced:** HEIC, HEIF, AVIF
- **Professional:** RAW (CR2, NEF, ARW, DNG, etc.), PSD
- **Vector:** SVG
- **Documents:** PDF

### Supported Output Formats
- WebP, PNG, JPEG, TIFF, BMP, ICO, PDF

## üöÄ USAGE

1. Launch PixBatcher
2. Click "Add Files" or drag & drop images into the window
3. Select output format from the dropdown menu
4. Choose destination folder (optional - defaults to source folder)
5. Adjust quality settings if needed
6. Click "Start Conversion"
7. Wait for the process to complete

## ‚öôÔ∏è CONFIGURATION

### Quality Settings
- **High Quality:** 90-100% - Minimal compression, larger file size
- **Medium Quality:** 70-89% - Balanced compression and quality
- **Low Quality:** Below 70% - Maximum compression, smaller file size

### Advanced Options
- **Preserve EXIF Data:** Keep original image metadata
- **Overwrite Existing Files:** Replace files with the same name
- **Multi-threading:** Utilize all CPU cores for faster processing
